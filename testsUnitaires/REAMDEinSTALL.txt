INSTALLER LE TEST SELENIUM: 
1.Telecharger composer pour PHP --> https://getcomposer.org/Composer-Setup.exe
2.Installé
3. Aller dans la console et aller dans le fichier où se trouve le composer installé(normalement dans users)
4. Créer un fichier qui s'appelle composer.json
5. Dans le composer.json mettre ceci 
{
    "require-dev": {
        "phpunit/phpunit": "7.8.0"
    }
 "require-dev": {
        "phpunit/phpunit-selenium": "3.2"
    }
}
6. Sauver le composer et fermer
7. Dans la console taper la commande: composer update
8. Les fichiers vont s'installer
9. Aller dans php storm-> settings->Test frameworks->php-> et puis cliquer + 
10. Ajouter phpUnit dans phpstorm et mettant le chemin vers autoload.php(autoload se trouve dans le fichier où le composer a été installé et c'est dans le fichier vendor)
11.Installer chromedriver.exe sur le pc
12. Installer serveur selenium
13. Mettre le fichier .jar et chromedriver.exe dans le même fichier et executer par la console 
14. La commande se trouve dans le README au niveau des tests Unitaires
